##Commands to remember

###Normal build commands
`python setup.py clean;`
`python setup.py test;`
`python setup.py build;`
`python setup.py install;`

###Test with tox
`virtualenv venv;`
`source venv/bin/activate;`
`tox;`
`deactivate;`
